    Icy Monitor

    Monitor your PC using your Android device!

    Download

    • Play Store Link
    • Windows Server

    Features:

    • Get graphs about your CPU (load, temperature, clock, power), GPU (load, clock, temperature, fan) and system (fans, temperature, voltages).
    • Customize graph colors to your liking.
    • Get information about each filesystem and each process running.
    • Save and monitor multiple computers effortlessly.
    • Clean, beautiful interface with tablet support.

    Setup

    Download the server application using the link above, extract all files in a folder and run the app as administrator (required to access sensor data). It is essential to open a port in the firewall (For windows firewall it can be done throught the app) for the server to work correctly. If you want to access the server through the Internet you will have to setup port forwarding.
    If you are updating from versions 1.4 and lower it is recommended to delete all old files.

    Libraries

    Computer Monitor uses HoloColorPicker by LersWerkman. HoloCircularProgressBar.java is based on CircularProgressBar by Pascal Welsch. The server used the OpenHardwareMonitorLibrary.
